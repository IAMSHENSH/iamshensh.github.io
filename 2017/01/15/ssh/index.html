SSH Server

Posted on 2017-01-15

ssh相关使用记录

环境

Ubuntu 16.10

安装

Ubuntu默认并没有安装ssh服务，如果通过ssh远程连接到Ubuntu，需要自己手动安装ssh-server。

  1. 判断是否安装ssh服务：
    ps -e|grep ssh
    系统返回如下既有ssh服务
    1172 ? 00:00:00 sshd
  2. ssh-server:
    ssh服务，安装后可被ssh远程登录
    sudo apt-get install openssh-server
  3. ssh-client:
    ssh终端，安装后可远程登录安装了ssh-server的机器
    sudo apt-get install openssh-client
    ssh服务启动后，即可登陆，登陆命令格式为：
    ssh 帐号@IP地址 例如：ssh test@192.168.135.249

工具

  1. PuTTY
    小巧，方便
  2. SecureCRT
    收费，不好看

  3. Xshell
